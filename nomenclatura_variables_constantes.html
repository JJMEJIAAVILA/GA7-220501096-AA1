Nomenclatura de Variables y Constantes

Importancia de una Buena Nomenclatura

Elegir nombres descriptivos y consistentes para variables y constantes es crucial para la claridad y el mantenimiento del código. Una buena nomenclatura ayuda a entender el propósito y el uso de cada elemento sin necesidad de leer comentarios extensos.

Principios Generales

  • Ser descriptivo y claro.
  • Utilizar un lenguaje consistente (generalmente inglés en la programación).
  • Evitar abreviaturas ambiguas.
  • Seguir convenciones específicas del lenguaje o del proyecto.

Ejemplos de Nomenclatura

Variables

  • Camel Case (JavaScript, Java): nombreUsuario, contadorDeElementos
  • Snake Case (Python, Ruby): nombre_usuario, contador_de_elementos
  • Kebab Case (CSS): ancho-del-contenedor, color-de-fondo

Constantes

  • Mayúsculas con Guiones Bajos (JavaScript, Python): PI, MAX_USUARIOS
